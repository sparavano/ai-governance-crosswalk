# ISO/IEC 42001 ↔ NIST AI RMF — Crosswalk (working document)

A practical mapping between ISO/IEC 42001:2023 (AI management system) main clauses and the NIST AI Risk Management Framework 1.0 functions (GOVERN, MAP, MEASURE, MANAGE), with commentary from a governance/GRC practitioner's perspective.

## Why

Organizations adopting both instruments need to see where an AIMS clause supports (or falls short of) an AI RMF outcome, and vice versa. Public clause-level crosswalks are scarce.

## Scope and method

- Mapping at ISO/IEC 42001 main-clause level (Clauses 4–10) against NIST AI RMF functions and categories.

- References are by clause/category number only. No text from ISO/IEC 42001 is reproduced (the standard is copyrighted). NIST AI RMF 1.0 is publicly available.

- Working document: Annex A control-level mapping and an ISO/IEC 27001:2022 layer are planned next.
